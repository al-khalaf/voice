Voice

A votebox for letting users request features.

View the Project on GitHub jarodl/voice

  • Download ZIP File
  • Download TAR Ball
  • Fork On GitHub

Setup

pip install voice

In settings.py:

INSTALLED_APPS = (
    ...
    'voice',
)

In urls.py:

urlpatterns = patterns('',
    (r'^$', include('voice.urls', namespace='voice')),
)

For further usage, look at the provided example_project.
